# Lunes Multi Keepalive

Multi-account keepalive for Lunes Host.

## Secrets

| Name | Description |
|------|-------------|
| `USERS_JSON` | `[{"username":"your@email.com","password":"yourpass"}]` |
| `LUNES_SERVER_URL` | Keepalive dashboard URL, e.g. `https://betadash.lunes.host/...` |
| `TG_BOT_TOKEN` | Telegram bot token (optional) |
| `TG_CHAT_ID` | Telegram chat ID (optional) |
